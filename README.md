# Abyss

## 说明

`Abyss`是`Android`平台下应用级系统调用拦截框架。

详细的文档可参考：

[【Abyss】Android平台应用级系统调用拦截框架](https://www.iofomo.com/blog/opensrcabyss)

## 快速开始

本项目在以下开发环境中编译通过：

-   Android Studio：4.2
-   Gradle：6.9.2
-   CMakeLists：3.6.0
-   NDK：21.4.7075529

```shell
$ ./gradlew build
```
